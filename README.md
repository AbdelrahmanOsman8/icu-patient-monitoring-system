# ICU Patient Monitoring System

## Overview
This Python project simulates a basic ICU patient monitoring system. It checks vital signs such as oxygen saturation and heart rate and provides alerts for abnormal or critical conditions.

## Objectives
- Monitor ICU patients’ oxygen levels and heart rate
- Alert for low oxygen, high heart rate, or critical combinations
- Provide quick feedback for healthcare staff to act on emergencies

## Features
- Monitor multiple patients at once
- Check vital signs against safety rules
- Display stable, warning, or emergency messages

## Technologies
- Python 3

## Healthcare Relevance
Monitoring ICU patients’ vital signs is critical for timely interventions. This system demonstrates how programming can support Clinical Decision Support Systems (CDSS) for critical care management.

## Future Improvements
- Integrate with real-time medical devices
- Store patient vitals in a database (SQLite)
- Include notifications (SMS/email) for emergencies
- Add multi-parameter scoring (like NEWS2 or MEWS)
